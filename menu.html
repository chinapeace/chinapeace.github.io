余额怎么查，怎么看有多少支票？

余额：curl localhost:1635/chequebook/balance | jq

支票：curl localhost:1635/chequebook/cheque | jq

所有查看余额、支票获取等操作访问：https://docs.ethswarm.org/docs/advanced/swap

bee-clef 的重启命令全部是什么？

假设clef脚本名为：clef-service

启动：./clef-service start

停止：./clef-service stop 或者CTRL +C

后台启动: nohup ./clef-service start &

Swarm bee钱包或者bee-clef地址私钥在哪里？

bee-clef私钥导出脚本： https://github.com/ethersphere/bee-clef/blob/master/packaging/bee-clef-keys

bee私钥导出脚本：搜索微信公众号"亚乐技术圈" 获取

没有节点连接上 要怎么排查下?

首先使用命令curl -s http://localhost:1635/peers | jq '.peers | length' 查看是不是连接上其他节点

其次观察bee的启动日志进行排查.

请问linux启动Bee后怎么后台运行？

cd /var/lib/bee

vi mypass.txt

输入自己的bee密码保存

找到一个位置执行：比如我在/root下执行:

nohup bee start --verbosity 5 --swap-endpoint https://rpc.slock.it/goerli --debug-api-enable --clef-signer-enable --clef-signer-endpoint /var/lib/bee-clef/clef.ipc --password-file /var/lib/bee/mypass.txt &

有查看多少节点在线的浏览器吗？

访问https://docs.ethswarm.org/debug-api/ 使用查节点的api访问

操作api提供：

curl http://localhost:1635/peers -确保已连接到群集

curl http://localhost:1635/balances-看到正在与同级交换数据

curl http://localhost:1635/settlements -查看其中任何余额是否超过了要结算的阈值。

curl http://localhost:1635/chequebook/cheque 查看是否有支票。 lastReceived代表可以兑现！

curl http://localhost:1635/addresses 获取自己的钱包地址，公钥信息
